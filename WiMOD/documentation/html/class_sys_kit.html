Detailed Description

Main class representing the interface to the WiMOD running the firmware WiMODLR-BASE.

Type definition for callback that handles a 'RX U-Data event' for the DataLink service Type definition for callback that handles a 'TX U-Data event' for the DataLink service Type definition for callback function that handles a 'RX U-Data event' for the DataLink service Type definition for callback function that handles a 'TX U-Data event' for the DataLink service This class is the only API class a user should use for interacting with a WiMOD module that runs the IMST WiMODLR-Base firmware.
